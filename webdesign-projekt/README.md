# Webdesign Projekt

Dieses Projekt ist eine einfache Webanwendung, die eine Webseite mit einem RGB-Hintergrund und einer transparenten Box in der Mitte enthält. Die Box kann verwendet werden, um Inhalte anzuzeigen.

## Projektstruktur

```
webdesign-projekt
├── src
│   ├── index.html        # Haupt-HTML-Datei der Webseite
│   ├── styles
│   │   └── main.css      # CSS-Stile für die Webseite
│   └── scripts
│       └── app.js        # JavaScript für Interaktivität
├── package.json           # npm Konfigurationsdatei
└── README.md              # Dokumentation des Projekts
```

## Installation

1. Klone das Repository oder lade die Dateien herunter.
2. Navigiere in das Projektverzeichnis.
3. Installiere die Abhängigkeiten mit dem Befehl:
   ```
   npm install
   ```

## Nutzung

- Öffne die `src/index.html` Datei in einem Webbrowser, um die Webseite anzuzeigen.
- Die transparente Box in der Mitte kann mit JavaScript interaktiv gestaltet werden, indem du die `src/scripts/app.js` Datei bearbeitest.
